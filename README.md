# 📌 Project Overview

This project analyzes global happiness trends using the World Happiness Report dataset covering 165 countries and 13 key indicators. The objective was to understand what drives happiness across countries and uncover patterns that can support data-driven insights on global wellbeing.

The analysis focuses on key variables such as GDP per capita, life expectancy, freedom, trust, and generosity, and how they influence overall happiness scores.

# 🎯 Objectives
1. Identify the top and bottom performing countries in terms of happiness
2. Analyze the relationship between happiness and key socio-economic factors
3. Explore patterns and disparities across countries
4. Present insights using clear and simple Excel visualizations
   
# 🛠️ Tools & Technologies
- Microsoft Excel – Data analysis and visualization
- Power Query (Excel) – Data cleaning and transformation
- VLOOKUP – Data merging and enrichment

# 🧹 Data Cleaning & Preparation (Power Query)

The dataset required significant cleaning before analysis. Using Power Query, I:

- Removed duplicate records to ensure data accuracy
- Handled missing or null values appropriately
- Renamed columns for clarity and consistency
- Ensured correct data types (e.g., numeric fields for GDP and scores)
- Standardized country names and indicators

This step ensured the dataset was clean, structured, and ready for analysis.

# 🔗 Data Transformation (Excel + VLOOKUP)

To enrich and structure the dataset:

- Used VLOOKUP to merge related data from different tables
- Combined indicators into a single structured dataset
- Verified accuracy of merged data to avoid mismatches

This allowed for more comprehensive analysis across multiple variables.

# 📈 Key Insights
1. Countries in Western Europe and North America consistently rank highest in happiness
2. There is a strong positive correlation between GDP per capita and happiness
3. Health (life expectancy) significantly contributes to higher happiness scores
4. Freedom to make life choices is a key driver of wellbeing
5. A significant gap exists between the happiest and least happy countries, with top countries scoring nearly three times higher
6. Countries such as Switzerland, Iceland, and Denmark rank highest, while Togo, Burundi, and Syria rank lowest

# ⚠️ Limitations
- The data is self-reported, which may introduce bias
- Important contextual factors (e.g., political stability, culture, conflict) are not included
- Some regions may have sampling limitations, affecting accuracy
- Certain calculated fields may be difficult to interpret without deeper statistical context

# 💡 Conclusion
This project demonstrates that economic stability, health, and personal freedom are major contributors to happiness. It also highlights how data analysis using Excel can uncover meaningful insights and support better understanding of global trends..
